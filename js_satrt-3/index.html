Переменные и константы в JavaScript

Ключевые слова let, const, var

Урок№3

Ключевое слово let для объявления переменной. Имя переменой может содержать только буквы, лутче латинского алфавита цифры, символы $ и _. При этом первый символ не должен быть цыфрой.

>>ОБЪЯВЛЕНИЕ ПЕРЕМЕННОЙ << и присвоение значения

let myLIfeStile = 'freelance';

Объявление без значения

let myLifeStile;

Присвоение значения

myLifeStile = 'freelance';

Верное объявление

let age;
let info123;
let $size;
let _color;

Неверное объявление

let 123 info;
let my - age;

ИЗМЕНЕНИЕ ЗНАЧЕНИЯ ПЕРЕМЕННОЙ

let myAge = 36;
myAge = 18;
console.log(myAge)
